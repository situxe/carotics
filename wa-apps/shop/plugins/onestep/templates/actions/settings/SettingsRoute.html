Статус плагина для поселения

Чтобы плагин работал для выбранного поселения необходимо задать "ВКЛ"

Режим
- корзина и оформление заказа

Плагин работает вместо стандартной корзины и всех шагов оформления заказа

- только оформление заказа

Плагин работает вместо оформления заказа, стандартная страница корзины плагином не изменяется

Только для ПК

Плагин работает только для настольных компьютеров и отключается для мобильных устройств.
Для мобильных работает стандартная корзина.

Шаблон

Шаблон темы дизайна, который использует плагин для вывода информации

Адрес страницы

Адрес страницы, по которому будет работать плагин.
Плагин не работатет с системными адресами такими, как cart/ и checkout/
